# NewsWise AI - News Article Classifier

Classifies news articles into **World**, **Sports**, **Business**, or **Sci/Tech** categories using TF-IDF + Logistic Regression.

- **Dataset:** AG News (120,000 training articles)
- **Accuracy:** 92.18% on test set
- **Model:** TF-IDF Vectorizer (50K features) + Logistic Regression
- **Built by:** Shubh Dave | Northeastern University | EAI 6010
